Sobre ::: API Cliente - Cotação de Ações

Esse é um projeto de estudo para consumo de dados de cotação de ações na bolsa de valores através de uma API (Alpha Vantage), tratamento e exibição das respectivas informações em uma página contendo tabela e gráficos. Desenvolvido em HTML5, CSS3, JavaScript e Google Charts.
